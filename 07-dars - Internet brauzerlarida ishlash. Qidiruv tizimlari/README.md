# 07-dars — Internet brauzerlarida ishlash. Qidiruv tizimlari

**Kurs:** Kompyuter savodxonligi · «IT Shaharcha»  ·  **Hajmi:** 2 akademik soat

## Mavzu haqida qisqacha
Brauzer va qidiruv tizimlarining ishlash tamoyili bilan tanishtirish; kalit so'zlar yordamida samarali qidiruv ko'nikmasini shakllantirish.

## Papka tarkibi
- **07-dars - Slayd.pptx** — darsda namoyish uchun brendlangan taqdimot.
- **07-dars - Qollanma.pdf** — o'qituvchi uchun to'liq qo'llanma (nazariya, atamalar, topshiriq, savollar).
- **07-dars - Topshiriq.pdf** — o'quvchi uchun alohida amaliy topshiriq varaqasi.

## Amaliy topshiriq
**"Tumanim bo'ylab raqamli sayohat"** — Internet — ulkan kutubxona, faqat to'g'ri so'rov berishni bilish kerak. Bugun sen o'z tumaningni "kashf" qilasan.

## Uyga vazifa
O'zingizni qiziqtirgan mavzu bo'yicha 3 ta ishonchli saytni topib, xatcho'pga saqlang.
