# 18-dars — Shartli formatlash, filtrlash, saralash amallari va himoya

**Kurs:** Kompyuter savodxonligi · «IT Shaharcha»  ·  **Hajmi:** 2 akademik soat

## Mavzu haqida qisqacha
Excel'da shartli formatlash, ma'lumotlarni saralash va filtrlash hamda jadvalni himoyalash ko'nikmalarini shakllantirish.

## Papka tarkibi
- **18-dars - Slayd.pptx** — darsda namoyish uchun brendlangan taqdimot.
- **18-dars - Qollanma.pdf** — o'qituvchi uchun to'liq qo'llanma (nazariya, atamalar, topshiriq, savollar).
- **18-dars - Topshiriq.pdf** — o'quvchi uchun alohida amaliy topshiriq varaqasi.

## Amaliy topshiriq
**"Rangli signalli jadval"** — Yaxshi jadval ma'lumotni o'zi "gapiradi". Bugun sen jadvalingga rangli signallar qo'shasan.

## Uyga vazifa
O'z baholaringiz jadvalini tuzib, 5 baholarini yashil, 3 baholarini qizil rangda shartli formatlang.
