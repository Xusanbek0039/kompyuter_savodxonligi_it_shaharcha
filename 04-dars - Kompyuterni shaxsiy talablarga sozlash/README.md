# 04-dars — Kompyuterni shaxsiy talablarga sozlash

**Kurs:** Kompyuter savodxonligi · «IT Shaharcha»  ·  **Hajmi:** 2 akademik soat

## Mavzu haqida qisqacha
Ish stoli foni, masalalar paneli, soat va sana sozlamalarini shaxsiy talablarga moslab o'zgartirish ko'nikmasini shakllantirish.

## Papka tarkibi
- **04-dars - Slayd.pptx** — darsda namoyish uchun brendlangan taqdimot.
- **04-dars - Qollanma.pdf** — o'qituvchi uchun to'liq qo'llanma (nazariya, atamalar, topshiriq, savollar).
- **04-dars - Topshiriq.pdf** — o'quvchi uchun alohida amaliy topshiriq varaqasi.

## Amaliy topshiriq
**"Kompyuterim — mening uslubim"** — Ish joyini o'ziga moslay olgan odam ko'proq zavq bilan ishlaydi. Bugun sen tizimni o'zingga moslaysan.

## Uyga vazifa
O'zingizga qulay bo'lgan mavzu (Theme) tanlab, kompyuter ko'rinishini to'liq sozlang.
