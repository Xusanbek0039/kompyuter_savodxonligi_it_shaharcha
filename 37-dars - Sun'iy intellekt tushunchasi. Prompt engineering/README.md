# 37-dars — Sun'iy intellekt tushunchasi. Prompt engineering

**Kurs:** Kompyuter savodxonligi · «IT Shaharcha»  ·  **Hajmi:** 2 akademik soat

## Mavzu haqida qisqacha
Sun'iy intellekt (SI) tushunchasi va prompt engineering asoslari bilan tanishtirish; SIga to'g'ri so'rov (prompt) berish ko'nikmasini shakllantirish.

## Papka tarkibi
- **37-dars - Slayd.pptx** — darsda namoyish uchun brendlangan taqdimot.
- **37-dars - Qollanma.pdf** — o'qituvchi uchun to'liq qo'llanma (nazariya, atamalar, topshiriq, savollar).
- **37-dars - Topshiriq.pdf** — o'quvchi uchun alohida amaliy topshiriq varaqasi.

## Amaliy topshiriq
**"SIga o'rgat va sina"** — To'g'ri savol — to'g'ri javobning yarmi. Bugun sen sun'iy intellektga savol berish san'atini sinaysan.

## Uyga vazifa
SIga sevimli mavzungiz bo'yicha 10 ta savolli viktorina tuzdirib, o'zingizni sinang.
