# Kompyuter savodxonligi kursi — O'quv materiallari

**«IT Shaharcha» — Yoshlar Axborot Texnologiyalari Markazi**

Ushbu to'plam "IT-Shaharcha"larda tashkil etiladigan **Kompyuter savodxonligi kursi** o'quv dasturi asosida tayyorlangan. Kurs jami **39 ta darsdan** (har biri 2 akademik soat) iborat.

## To'plam tuzilishi

Har bir dars uchun alohida papka tayyorlangan. Har bir papka ichida:

- **Slayd.pptx** — darsda namoyish qilish uchun brendlangan taqdimot;
- **Qollanma.pdf** — o'qituvchi uchun to'liq qo'llanma (nazariy qism, atamalar, amaliy topshiriq va nazorat savollari);
- **Topshiriq.pdf** — o'quvchi uchun alohida, har bir mavzuga mos qiziqarli amaliy topshiriq varaqasi;
- **README.md** — mavzu bo'yicha qisqa ma'lumot.
